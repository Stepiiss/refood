ReFood

ReFood je webová aplikace v Reactu (Vite + Firebase), která propojuje uživatele pro sdílení přebytků jídla. Umožňuje registraci a přihlášení, přidávání nabídek, filtrování nabídek, zobrazení detailu produktu a mapové zobrazení nabídek.

Návod k instalaci

Lokální spuštění

  • Nainstalujte Node.js 20.19 nebo novější.
  • Naklonujte repozitář: git clone https://github.com/Stepiiss/refood.git
  • Přejděte do projektu: cd refood
  • Nainstalujte: npm install
  • Vytvořte soubor .env a nastavte VITE_GOOGLE_MAPS_API_KEY.
  • Spusťte: npm run dev

Nasazení na server

  • Nastavte VITE_BASE_PATH podle cesty aplikace (např. /dist/).
  • Vytvořte produkční build: npm run build.
  • Nahrajte obsah složky dist/ na server.
  • Nastavte SPA fallback na index.html pro routy jako /offers.
